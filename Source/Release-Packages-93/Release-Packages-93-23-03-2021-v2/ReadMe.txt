Release note:

- Contains update for Publishing as well as Indexing
- Built on .NET 4.7.2
- Sitecore 9.3 Compatible
- Sitecore Publishing Service Compatible
- Sitecore Normal Publishing Compatible

Example Commands:

**Publishing:**
FortisCollections.Toolcore.Publish.Runner.exe -s "master" -u "YOURHOST" -r "/sitecore" -d

**Indexing:**
FortisCollections.Toolcore.Indexing.Runner.exe -u "YOURHOST" -i "sitecore_sxa_web_index"
